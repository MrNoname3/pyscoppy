# pyscoppy

A host-side USB driver for a **shared** Scoppy oscilloscope: one daemon owns the
serial link so a human (browser GUI) and an AI agent drive the same live scope at
once. If you're an agent working here, the operating guide below — how to start a
session, what you can set vs. what only the human can, and how to read the data —
is the canonical reference; read it before acting.
